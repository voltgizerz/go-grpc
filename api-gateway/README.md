## API Gateway
- Connect Order Service and User Service and Auth Service

## SET ENV
```sh
ORDER_GRPC=localhost:3000
USER_GRPC=localhost:3001
PORT=1000
```

## ENDPOINT
```sh
Auth
- POST api/register (on going)
- POST api/login (on going)
- POST api/validate (on going)

Order
-GET api/orders

User
-GET api/users
```
