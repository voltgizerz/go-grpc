## Example Simple Go Microservices gRPC
- Auth Service
- Order Service
- User Service
- API Gateway

## Install

```sh
1. Go version 1.17.8 or higher
```

## Usage

```sh
1. Set .env
2. Run all gRPC server
3. Run API Gateway
4. Test using postman.
```
