# Simple Go Microservices gRPC
Simple Microservices using Go with faker data.
- API Gateway
- Auth Service
- Order Service
- User Service

# BUILD WITH
- Chi
- Faker

# Install

```sh
1. Minimum Go version 1.17.8 or higher installed.
```

# How to run project?

```sh
1. Set .env
2. Run all gRPC server
3. Run API Gateway
4. Test request using postman
```
